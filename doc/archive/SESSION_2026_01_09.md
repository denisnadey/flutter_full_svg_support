## Technical Details

### Syncbase Timing Features

**Syntax Supported:**
```xml
<!-- Simple sync -->
<animate begin="anim1.begin" ... />

<!-- End sync with offset -->
<animate begin="anim1.end+2s" ... />

<!-- Repeat sync -->
<animate begin="anim1.repeat(2)" ... />

<!-- Multiple conditions -->
<animate begin="2s; anim1.end+1s; click" ... />
```

**Implementation Highlights:**
- Topological sort for dependency resolution
- Circular dependency detection
- Graceful fallback for missing references
- O(n) resolution at timeline creation
